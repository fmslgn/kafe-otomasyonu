# Stitch Tasarım Uygulama Notları

Bu klasördeki görseller, Kafe Otomasyonu Flutter projesinin modern UI/UX tasarım referanslarıdır.

## Genel Kurallar

- Tüm arayüz metinleri Türkçe kalmalıdır.
- Mevcut çalışan işlevler bozulmamalıdır.
- Flutter sadece ApiService üzerinden backend API kullanmalıdır.
- Backend, PostgreSQL ve endpoint yapısı gereksiz değiştirilmemelidir.
- Yönetici tarafından seçilen tema rengi korunmalıdır.
- Logo varsa ilgili ekranlarda gösterilmelidir.
- QR menü dikey/yatay görünüm ayarı bozulmamalıdır.
- Prim sistemi kapalıysa ilgili prim kartları görünmemelidir.
- Kurye prim sistemi kapalıysa kurye prim kartı görünmemelidir.
- Popup, tooltip, hata ve boş veri tasarımları korunmalı veya modernleştirilmelidir.
